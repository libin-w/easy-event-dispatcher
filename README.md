# easy-event-dispatcher

## Installing

Using npm:

```bash
$ npm install axios
```

## Methods

| 方法名  | 说明     | 参数                   |
| ------- | -------- | ---------------------- |
| trigger | 触发事件 | eventName:string       |
| on      | 注册事件 | eventName:string, func |
| off     | 移除事件 | eventName:string, func |
